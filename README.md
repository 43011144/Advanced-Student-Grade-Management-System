# Advanced Student Grade Management System

An advanced desktop-based **Student Grade Management System** developed in **Python** using **Tkinter**. The application provides a graphical user interface (GUI) for capturing student assessment marks, automatically calculating weighted grades, assigning grade symbols, managing student records, validating user input, and executing automated unit tests.

Student records are stored using CSV files, making the application lightweight while demonstrating practical software engineering concepts such as GUI development, modular programming, data validation, file handling, and software testing.

---

## Overview

This project was developed to demonstrate the implementation of a complete desktop application using Python.

The system enables users to:

- Capture student assessment marks
- Automatically calculate weighted final grades
- Assign grade symbols
- Store student records in CSV format
- View all student records
- Search for students
- Update existing records
- Delete student records
- Execute automated unit tests

## Installation

### Clone the repository

```bash
git clone https://github.com/43011144/advanced-student-grade-management-system.git
```

### Navigate to the project

```bash
cd advanced-student-grade-management-system
```

### Run the application

```bash

```

No external libraries are required. The application uses only Python's built-in modules.
